# 💰 Smart Personal Finance Dashboard

## 📌 Project Overview

The **Smart Personal Finance Dashboard** is a full-stack web application designed to help users track expenses and gain actionable financial insights through AI.

This project was built using an **AI-first development approach**, leveraging Cursor Composer to generate more than 90% of the codebase. The focus is on **delegation, prompt engineering, and iterative refinement**, rather than manual coding.

---

## 🎯 Objectives

* Track personal expenses efficiently
* Visualize spending patterns with charts
* Provide AI-driven financial insights
* Demonstrate AI-assisted full-stack development
* Deliver a production-ready deployed application

---

## 🚀 Tech Stack

* **Frontend:** Next.js (App Router) + Tailwind CSS
* **Backend:** Next.js API Routes
* **Charts:** Chart.js
* **Icons:** Lucide React
* **Deployment:** Vercel
* **IDE:** Cursor (AI-powered development)

---

## 🧠 Key Features

### 📊 Expense Tracking

* Add, edit, and delete expenses
* Categorize expenses (food, rent, transport, etc.)
* Real-time updates

### 📈 Data Visualization

* Total spending overview
* Spending by category (Pie Chart)
* Spending over time (Line Chart)

### 🤖 AI Insights

* Detect overspending patterns
* Suggest savings opportunities
* Provide category-based recommendations

### 🎨 Modern UI/UX

* Dark mode interface
* Glassmorphism design (blur + transparency)
* Responsive layout for all devices
